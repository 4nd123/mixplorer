# Fonts

Font files must be in '.ttf' (TrueType Font) format. for other formats are not supported.

1. Download the font you want
2. Copy fonts 'font-name.ttf' to 'fonts/font-name.ttf'
3. Open and add this code to on 'properties.xml'

  ```xml
  <entry key="font_primary">fonts/fontname.ttf</entry>
  <entry key="font_secondary">fonts/fontname1.ttf</entry>
  <entry key="font_title">fonts/fontname2.ttf</entry>
  <entry key="font_popup">fonts/fontname3.ttf</entry>
  <entry key="font_editor">fonts/fontname4.ttf</entry>
  <entry key="font_hex">fonts/fontname5.ttf</entry>
  ```

For more see <https://forum.xda-developers.com/t/tutorial-mixplorer-themes-skins-how-to-make-them.4202319/>
